How to build in 4 Steps

Step 1

Open the Developer Command Prompt or if you will use Visual Studio you can run the scripts directly.

Step 2

Run the download.bat script to download the App's Source Code from the github repository.

Step 3

Run the setup.bat and enter the Info , Certificate Pass is the path to the .p12 file found in the Tizen Certificate Manager.

Step 4

If you are in the Developer Command Prompt run the build.bat script from the Prompt and if you are using Visual Studio go to AppSourceCode and Open the GreentegCoreApp1.sln file.
